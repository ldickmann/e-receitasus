## 📄 Sobre o Projeto

Este repositório contém o início do desenvolvimento do **E-ReceitaSUS**, um aplicativo destinado à digitalização do processo de revalidação de receitas médicas.

Neste momento, o README mantém uma versão **simplificada**, servindo apenas como apresentação inicial da disciplina e do projeto.

Quando finalizarmos o desenvolvimento completo do frontend e backend, será criado um README final mais detalhado, documentando:

- Arquitetura completa do projeto;
- Tecnologias utilizadas;
- Metodologias aplicadas (TDD + Scrum);
- Estrutura de diretórios;
- Execução, testes e CI/CD.
